# Clipshare

A clipboard sharing tool written in Go.

## Usage

### Start the tool
  ```clipshare```

### Get contents of clipshare buffer (received from other peers)
  ```clipshare get```

### Send clipboard contents to other peers
  ```clipshare set 127.0.0.1 10.0.2.1```
